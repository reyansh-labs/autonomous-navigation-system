# 🚗 Autonomous Robot Navigation
### Self-Driving Vehicle (SDV) Engineering Project

## About This Project

This project is part of my Self-Driving Vehicle (SDV) engineering program. My goal is to learn how autonomous vehicles sense the world, make decisions, and navigate without human control.

In this project, I programmed a robot to navigate a 10 × 10 ft challenge mat using **Time-of-Flight (ToF) sensors** and a **camera**. Along the way, I learned how robotics combines programming, engineering, sensors, and testing to solve real-world problems.

---

# Project Objectives

The objectives of this project are to:

- Build and program an autonomous robot
- Detect obstacles using sensors
- Navigate a challenge course without human control
- Understand how autonomous systems make decisions
- Improve the robot through testing and experimentation

# How the Robot Works

The robot continuously collects information from its sensors.

1. Detects obstacles using Time-of-Flight sensors
2. Uses the camera to understand its surroundings
3. Processes the sensor data
4. Makes navigation decisions
5. Controls steering and speed
6. Continues until it reaches the destination

This is similar to how real self-driving vehicles use sensors and software to drive safely.

# Engineering Design

I also learned how engineering design is an important part of robotics.

Using **OnShape CAD**, I designed parts of the robot chassis before creating them with a **3D printer**.

Designing, testing, and improving the robot taught me that engineering is an iterative process. Small changes can make a big difference in how well the robot performs.

---

# Challenges

One of the biggest challenges was making the robot navigate consistently.

Sometimes the robot:

- Turned too early
- Missed an obstacle
- Didn't follow the expected path

To improve it, I tested different values, adjusted the code, and learned from each attempt until the robot performed better.

This taught me that failures are part of engineering and every test helps improve the final solution.
